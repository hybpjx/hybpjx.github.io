安装Java环境

选择 自己相应的 版本 一键式安装接口

添加环境变量

  • 下载 .dmg 文件并安装，默认路径为：
    /Library/Java/JavaVirtualMachines/zulu-8.jdk/Contents/Home

  • 添加至 jenv 管理

    • jenv add /Library/Java/JavaVirtualMachines/zulu-8.jdk/Contents/Home
jenv versions  # 验证是否显示 "1.8"
