安裝 uv

uv 本身並不需要 Python，所以不建議用 pip 或是 pipx 安裝，這樣都會跟特定的 Python 環境綁在一起，Windows 上就直接透過 PowerSehll 安裝即可：

scoop 安裝：

scoop install uv
本文都以 Windows 平台在 PowerShell 下操作示範
