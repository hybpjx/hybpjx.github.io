启动 Notebook

1、打开终端（Terminal）

2、进入你想存文章的文件夹，例如：

cd ~/Documents/article_project

3、启动 Jupyter

jupyter notebook

会自动打开浏览器，显示文件列表

点击 New → Python 3 新建 Notebook
