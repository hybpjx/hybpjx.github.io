ubuntu安装anconda3 与Miniconda

2022-03-15 快速安装 python / ubuntu

ubuntu 安装anaconda3

官网:https://www.anaconda.com/
下载:https://www.anaconda.com/products/individual#Downloads
MiniAndonda: https://docs.anaconda.com/miniconda/

安装

输入以下命令执行所下载的安装程序：

Miniconda

使用

为了下载库的时候速度更快，更换conda下载源为清华大学镜像

在 /home/xxx里面添加Anaconda-python路径

然后在ubuntu 输入命令

conda –version

建立虚拟环境

conda create -n test python=3.9

激活 虚拟环境

conda activate test

在虚拟环境安装软件

conda install spyder

关闭虚拟环境

deactivate

移除环境

conda remove –name test –all

删除已经设定好默认镜像源，恢复原来的源

conda config –remove-key channels

列出当前所有的环境

conda env list

重置Base环境

conda install –revision 0
