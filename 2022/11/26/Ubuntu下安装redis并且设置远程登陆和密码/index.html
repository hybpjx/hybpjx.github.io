Ubuntu下安装redis并且设置远程登陆和密码

hybpjx
2022-11-26
快速安装
redis, ubuntu

安装redis

sudo apt-get install -y redis-server

更改配置

sudo vim /etc/redis/redis.conf



如果不知道怎么找
直接在命令行模式下输入
/{你要搜索的东西} 然后按回车 然后按n是下一个 N是上一个

重启服务

systemctl restart redis-server.service
